# Cookbook

Each recipe below starts with the example output and shows the exact command that produced it. Copy, paste, edit the `--prompt`, and regenerate.

All example artifacts live in [`assets/examples/`](https://github.com/ianlintner/foundry-image-gen/tree/main/assets/examples) in the repo and are regenerated by CI against the bundled QA gates.

## Hard vs. soft gates

Every recipe supports `--qa`. Hard gates (palette fidelity, alpha crispness, seam continuity, silhouette IoU) exit non-zero on fail. Soft gates (palette coverage, outline coverage) warn to stderr but still exit `0`. See [QA metrics](../reference/qa.md) for the full list.
